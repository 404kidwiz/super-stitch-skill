# Clone Reference Example

Use this when you want a high-fidelity baseline before redesigning.

```text
Use super-stitch-skill to recreate this site as closely as possible in Google Stitch.

Inputs:
- Live URL: https://example.com
- Priority: desktop-first
- Also capture mobile behavior

Requirements:
- Start with a clone audit
- Identify must-match elements vs simplifiable elements
- Write the first-pass Stitch clone prompt
- After generation, return a delta checklist for the next refinement round
- Do not redesign the page yet
```

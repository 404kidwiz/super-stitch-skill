# Redesign After Clone Example

Use this when the goal is not just fidelity, but a stronger finished product.

```text
Use super-stitch-skill to clone this landing page first, then branch into a premium redesign once the structure is close.

Redesign direction:
- Dark tech
- cinematic motion
- sharper CTA hierarchy
- cleaner card system

Keep:
- the overall information architecture
- the section order
- the core conversion path

Improve:
- typography scale
- spacing rhythm
- visual hierarchy
- mobile nav behavior
```

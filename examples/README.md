# Examples

These examples are meant to be copied into Gemini, Claude, Codex, OpenCode, or any other skill-aware CLI that can invoke `super-stitch-skill`.

Files:
- `clone-reference.md` for high-fidelity site recreation from a live URL or screenshots
- `redesign-after-clone.md` for preserving structure while upgrading the visual system
- `code-handoff.md` for turning an approved Stitch direction into frontend implementation guidance
